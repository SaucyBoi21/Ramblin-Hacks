# PK Prophet

A Ramblin' Hacks project created by Saahas Swaroop, Stephen Viray, Vivaan Rajesh, and Andrew Pauley.

## How to run on local machine

Windows:
```
python3 -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
MacOS/Linux:
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
